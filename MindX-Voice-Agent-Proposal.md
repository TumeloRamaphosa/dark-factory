## What We Are Building

A custom **Voice Agent** for MindX — an AI-powered conversational agent that:
- Speaks to clients via voice (real-time, natural dialogue)
- Lives in a sovereign South African VM (data stays in SA)
- Handles inquiries, qualifies leads, books appointments
- Available 24/7 — never sleeps, never takes a day off
- POPIA-compliant — client data stays in South Africa

Think of it as adding a world-class voice AI team member who knows your business inside out.

---

## The Technology

**Base:** RileyJarvis Architecture
- Electron + React + Vite + TypeScript
- OpenAI Realtime API (voice-to-voice)
- Web-based — no installation required for end users
- Persistent memory across sessions

**Hosting:** Orgo Cloud VM (South Africa)
- Sovereign SA compute
- Open floor / shared space deployment ready
- Accessible via web browser + phone

**Stack:**
- Frontend: React (web app)
- Voice: OpenAI Realtime API (your API key or ours)
- Backend: Node.js + Express
- Memory: Persistent conversation history
- Integrations: WhatsApp, web chat, phone
