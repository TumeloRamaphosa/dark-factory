Overview — The 18-Month Path

This study path transforms Tumelo from an emerging developer into a technical leader capable of building, scaling, and managing the LAISA Agent OS VM infrastructure business. The path combines Google Cloud certifications with practical agentic engineering skills. All courses are free unless noted.

Phase 1 — Foundation (Month 1-2)

1. Google Cloud Digital Leader

cloud.google.com/certification/cloud-digital-leader

Cloud fundamentals, products, pricing, security. 4-6 hours. Free exam.

2. Generative AI Fundamentals

cloudskillsboost.google — Foundational Generative AI

Gemini API, prompt design, grounding, RAG. 4 hours. Free.

Phase 2 — AI/ML Engineering (Month 2-4)

3. Introduction to AI/ML on Google Cloud

cloudskillsboost.google — Introduction to AI and ML

Vertex AI, pre-trained APIs (Vision, Language, Speech), AutoML. 6 hours. Free.

4. Vertex AI Agent Builder

cloudskillsboost.google — Vertex AI Agent Builder

Building RAG agents, MCP servers, Vertex AI Search, grounding with enterprise data. 6 hours. Free.

Phase 3 — Infrastructure Engineering (Month 3-5)

5. Google Cloud Computing Fundamentals

cloudskillsboost.google — Compute Essentials Quest

Compute Engine, Cloud Run, Kubernetes Engine (GKE), Cloud Functions, App Engine. 6 hours. Free.

6. Kubernetes in Google Cloud

cloudskillsboost.google/quest/193

GKE clusters, deployments, auto-scaling, Helm charts, multi-tenant clusters. 8 hours. Free.

7. Networking on Google Cloud

cloudskillsboost.google

VPC networks, load balancing, Cloud CDN, DNS, Private Service Connect. 6 hours. Free.

Phase 4 — Agentic Systems (Month 5-7)

8. Build AI Agents with LangChain + Vertex AI

python.langchain.com/docs/get_started + cloud.google.com/solutions/gen-ai

LangChain orchestration, tool calling, multi-agent systems, memory management, Vertex AI as LLM backend.

9. Responsible AI for Developers

cloudskillsboost.google — Responsible AI

AI bias, fairness, safety, evaluation, governance. Critical for healthcare AI. 4 hours. Free.

Phase 5 — Data Engineering (Month 6-9)

10. Professional Data Engineer (Study)

cloud.google.com/certification/data-engineer | $200 exam

BigQuery, Dataflow, Pub/Sub, Looker, ETL/ELT pipelines. Build: LAISA Analytics Pipeline — Supabase + BigQuery + Looker Studio.

11. Professional Machine Learning Engineer (Study)

cloud.google.com/certification/machine-learning-engineer | $200 exam

Vertex AI ML pipelines, feature engineering, training, MLOps. Build: Patient churn prediction model from Supabase data.
