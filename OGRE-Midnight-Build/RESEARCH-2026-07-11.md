## 🇺🇸 SECTION 1 — OPENCLAW & HERMES AGENTS

### What They're Building
- **OpenClaw** (openclaw.ai) is an open-source AI agent framework for self-hosted deployment. Teams ran 7 OpenClaw agents 24/7, shipping 10 production features in one month at ~$4K/week. It runs on private GPU infra, owns your data, picks your models. CMMC/HIPAA/ITAR ready. Pricing: $10–25/month for self-hosted infrastructure + model usage costs from free local models to premium API calls. Usage-based cost structure is the selling point.
- **Hermes Agent** (Nous Research) is an open-source autonomous agent that **gets more capable the longer it runs**. Built to run continuously — responds to requests, plans multi-step tasks, executes autonomously, and self-improves. Lives on a $5 VPS, GPU cluster, or serverless. NVIDIA highlighted it as a reference agent on RTX AI Garage. Memory persistence is its standout: it "remembers what it learned yesterday."

- **Cursor** → AI-first code IDE, Agent mode + Composer + .cursorrules, trusted by world-class teams
- **Codeium** → Free/cheap autocomplete, fast, good for individual devs
- **GitHub Copilot** → Enterprise-grade, deep VS Code/GitHub integration, Microsoft's moat
