## PART 5: VOICE AGENT — VOICEBOX INTEGRATION

### What We Can Build
Voicebox (https://github.com/jamiepine/voicebox) is a local-first, free, open-source AI voice studio. It runs entirely on your machine — no cloud, no subscription.

**What it can do for Maphiri Books:**

1. **Chapter Read-Alouds (YouTube)**
   - Clone Keamogetswe's voice from 30 seconds of audio
   - Generate audio for each chapter
   - Post to YouTube with Mapiri character visuals
   - Kids can listen to the story while following the book

2. **WhatsApp Voice Bot**
   - Parents opt in via WhatsApp
   - Send "Mapiri tip" to get daily financial literacy micro-lesson
   - Send "Chapter 1" to receive audio clip

3. **Audiobook Preview Clips**
   - 30-second clips per chapter, auto-generated
   - Post to TikTok, Instagram, YouTube Shorts

4. **School Virtual Visits**
   - Pre-recorded Q&A with "Mapiri's voice" (Keamogetswe's cloned voice)
   - Scale beyond in-person appearances

5. **Agent Voice for Marketing**
   - Keamogetswe records 30 seconds
   - All future audio content uses her voice
   - Consistent brand voice at scale

### Voicebox Setup Steps
```bash
# Install Voicebox
git clone https://github.com/jamiepine/voicebox.git
cd voicebox && npm install && npm run dev

# Clone Keamogetswe's voice
# 1. Record 30 seconds of Keamogetswe reading a chapter
# 2. Load into Voicebox → Voice Cloning → Create new voice "Keamo"
# 3. Generate TTS in English (can extend to Sesotho, Zulu)

# Connect to ORGO VM
# Voicebox has an MCP server built in
# Connect to OpenClaw on ORGO VM for full automation
```

### Voice Agent Pipeline (Automated)
```
Keamogetswe records 30s → Voicebox clones voice → Cipher Tr@ce writes script → 
Voicebox generates audio → Posted to YouTube + WhatsApp + TikTok
```

## PART 7: REMOTION SPHERE ANIMATION

### What It Is
A 5-second animated sphere in Mapiri brand colors for YouTube intros, website hero, and social clips.

### Visual Description
- **Background:** Deep navy `#1A1A2E` fading to dark
- **Sphere:** Floating, slowly rotating, gradient from hot pink `#E91E8C` to emerald green `#2ECC71`
